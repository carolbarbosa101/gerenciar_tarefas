# Task Manager

This is a project of a task manager developed using the following technologies:

- **Frontend:**
  - React
  - Bootstrap
  - Tailwind CSS

- **Backend:**
  - PHP Laravel
  - PostgreSQL

## Installation Guide

### 1. Setting up the Development Environment

Before getting started, ensure you have the following tools installed on your system:

- **Node.js and npm:** [Download and install Node.js and npm](https://nodejs.org/).
- **Composer:** [Download and install Composer](https://getcomposer.org/download/).
- **Git:** [Download and install Git](https://git-scm.com/downloads).

### 2. Cloning the Repository

Clone the project repository to your local machine using the following command:

git clone https://github.com/carolbarbosa101/gerenciar_tarefas.git
cd your-project
